# Project Context

## Overview
**Mr.Holmes** is an Open Source Intelligence (OSINT) tool focused on robust information gathering. It operates primarily as a Python-based CLI application with a supplementary PHP GUI for viewing generated reports.

**Project Type ID:** `cli`

## Target Audience
- OSINT Researchers
- Cybersecurity Analysts
- Ethical Hackers
- Private Investigators

## Core Business Logic & Boundaries
- **Objective:** Collect public datal leaks, and information footprint across multiple online sources.
- **Data Categories:** Usernames, Phone numbers, Websites/Domains, Persons, Ports, Emails, Google/Yandex Dorks.
- **Constraints & Policies:** Relies strictly on public data sources. Uses proxy configurations to anonymize requests. Implements logic to handle errors, rate limits, and blockages systematically.
- **Boundaries:** The application is split between a Python CLI for the heavy lifting (data gathering/scraping) and a PHP GUI component that visualizes the structured outputs (JSON/Text reports) generated by the Python core.
